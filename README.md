# Portfolio

A simple static portfolio website built with HTML, CSS and JavaScript.

## Demo
- Open the site by double-clicking [index.html](index.html) in your browser.
- For a local development server, run (Python 3):

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Or use VS Code's Live Server extension: right-click [index.html](index.html) → "Open with Live Server".

## Project Structure
- [index.html](index.html) — main page
- [css/stylesheet.css](css/stylesheet.css) — styles
- [scripts/nav.js](scripts/nav.js) — navigation script
- [pictures/](pictures/) — image assets

## Development
- Edit files in your editor and refresh the browser to see changes.
- To test on a simple HTTP server (recommended), use the Python command above.
